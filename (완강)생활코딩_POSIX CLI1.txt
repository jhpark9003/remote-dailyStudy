# 생활코딩_POSIX CLI1

POSIX : Portable Operating System Interface

         File            Directory
Create  editor              mkdir
Read    editor, cat, ls     ls
Update  editor, mv          mv
Delete  rm                  rm


pwd         : print working directory
/           : root directory
~           : home directory
cd          : change directory
--help      : simple manual
man command : manual
ls -l       : list in long format
touch       : make empty file
.filename   : hidden file
ls -a       : show all files
mkdir       : make directory
./          : current directory
mv          : move (rename)
rm          : remove file
rm -r       : remove directory
relative path vs absolute path
../         : parent directory
nano        : text editor
cat         : 간단히 보기
;           : command seperator
&&          : and operator


 - Program
정교하게 동작하는 프로그램을 만들고 싶다면
shell script를 찾아서 공부하자.
컴퓨터를 자동화해서 생산성을 비약적으로 향상시킬 수 있다.

 - Package
package manager 프로그램을 이용하면 CLI에서 명령어만으로 간편하게 프로그램 설치 가능
리눅스 : apt-get, yum
맥     : homebrew
윈도우 : chocolatey

 - Maintain
컴퓨터 : data와 processor로 이루어져 있다.
data  : storage와 memory
CLI에서 관리하는 프로그램이 top이고, 이것을 조금 더 보기좋게 관리하는 프로그램이 htop
컴퓨터 원리를 파악하려면 '컴퓨터 구조(computer architecture)'를 공부해보자.
